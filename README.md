# E-Commerce Buying Pattern Analysis

## 📌 Overview
This project analyzes customer buying patterns in an e-commerce dataset to uncover meaningful insights that help improve business decisions, marketing strategies, and customer engagement.

The analysis is performed using SQL, Python (Pandas), and Power BI, covering the complete workflow from data extraction to visualization.

---

## 🎯 Objectives
- Understand customer purchasing behavior  
- Identify trends in sales and product demand  
- Analyze customer segments and buying frequency  
- Generate actionable business insights  

---

## 🛠️ Tech Stack
- SQL → Data extraction & querying  
- Python (Pandas) → Data cleaning & analysis  
- Power BI → Data visualization & dashboards  
- Jupyter Notebook → Analysis workflow
